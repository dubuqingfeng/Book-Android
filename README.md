# BookBook    [![Google Play](http://developer.android.com/images/brand/en_generic_rgb_wo_45.png)](https://play.google.com/)

## 预览
####是一款服务于管理、查书、比价、购买备忘的移动应用。你可以：
* 用手机拍ISBN条码，快速找书
* 查询图书信息和评价
* 对比不同网上书店的价格，货比三家
* 随时增减购书单，并与网页同步
